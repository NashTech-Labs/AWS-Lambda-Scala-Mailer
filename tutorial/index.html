AWS-Lambda-Scala-Mailer

Building AWS lambda function with scala. This is a classic application with lambda fuction to send mail notification whenever some action perform with your AWS service.

Clone Project


        git clone https://github.com/abhisheknoldus/AWS-Lambda-Scala-Mail.git
        cd AWS-Lambda-Scala-Mail
        sbt assembly
Follow this blog ( https://blog.knoldus.com/2017/04/01/serverless-architecture-blog-2-aws-lambda/ ) to deploy this code to AWS lambda
For any issue please raise a ticket @ [Github Issue](https://github.com/knoldus/lagom-java-crud/issues)
